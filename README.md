# SDRCC (Self Driving Remote Control Car)
Implementation of code on a RC car

## 1. Implementation with USB cable range
Parts required : 
1. RC Car
2. 2 Camera
3. Raspberry pi
4. IMU
5. USB(3m)
6. GPS Sensor
7. Arduino 

For reference : https://zhengludwig.wordpress.com/projects/self-driving-rc-car/
